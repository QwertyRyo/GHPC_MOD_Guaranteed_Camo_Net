# Guaranteed Camo Nets for Gunner HEAT PC!
GHPC camo nets look great, but they don't spawn all of the time. This mod fixes that!

Alternatively if you hate vehicles hiding and don't want them to have camo nets, this mod also fixes that.

Version 1.1.0 of the Guaranteed Camo Nets in Gunner HEAT PC. It guarantees that camo nets will always spawn on vehicles, rather than some of the time, and guarantees all of the camo nets on the vehicle will spawn. In UserData/MelonLoaderPreferences.cfg, you can pick which vehicles will be guaranteed to receive camo nets by setting them to True or False - vehicles set to False will never spawn camo nets.

This mod is compatible with Pact Increased Lethality - you will however need to be aware of which base game vehicles map to the modded vehicles (i.e. T-80B -> T-80BV). There may also be clipping between the ERA and the camonets, as the camonets were not designed with PIL in mind. I may fix this in the future.

If you don't see a vehicle in the config file, it probably means that vehicle cannot mount camo nets. If you do see a vehicle which can spawn with a camo net that is not in the config file, please respond in Issues.


***Installation***  
This mod requires MelonLoader Version 0.6.1.
This mod does not require the Vehicle Preloader or Texture Loader.

To install, extract the "GuaranteedCamoNets.dll" file from releases and unzip the "GuaranteedCamoNets" assets folder into the /mods/ folder.
