# Guaranteed Camo Nets for Gunner HEAT PC!

Version 1.0.0 of the Guaranteed Camo Nets in Gunner HEAT PC. It guarantees that camo nets will always spawn on vehicles, rather than some of the time, and guarantees all of the camo nets on the vehicle will spawn. In UserData/MelonLoaderPreferences.cfg, you can pick which vehicles will be guaranteed to receive camo nets. 

Version 1.1.0 will add a feature allowing users to guarantee camo nets will NOT spawn on selected vehicles.

***Installation***  
This mod requires MelonLoader Version 0.6.1.
This mod does not require the Vehicle Preloader or Texture Loader.

To install, extract the "GuaranteedCamoNets.dll" file from releases and unzip the "GuaranteedCamoNets" assets folder into the /mods/ folder.
